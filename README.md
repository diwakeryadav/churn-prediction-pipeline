# Churn Prediction Pipeline

An end-to-end ML pipeline for customer churn prediction with automated data generation, ETL processing, feature engineering, and model training.

## Overview

This project implements a complete data-to-model pipeline using Apache Airflow, featuring:
- Synthetic data generation
- ETL (Extract, Transform, Load)
- Feature engineering
- Model training
- Automated workflow orchestration

## Project Structure

```
churn-prediction-pipeline/
├── dags/              # Airflow DAG definitions
├── data/              # Data generation and storage
├── etl/               # ETL scripts
├── features/          # Feature engineering
├── models/            # Model training and artifacts
└── README.md          # This file
```

## Getting Started

1. Install dependencies
2. Configure Airflow
3. Run the pipeline
